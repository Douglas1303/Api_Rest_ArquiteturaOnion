# Api_Rest_ArquiteturaOnion
Api rest em C# na arquitetura Onion.
Nesse projeto eu utilizo:

Dot Net Core 3.1, DDD, Linq, Solid, Cqrs, Entity framework, Code First, Dapper, Unit Of Work, Fluent Validation, Mediator, Xunit, Acesso a api ViaCep.

O objetivo desse projeto é desenvolver uma solução que atenda o nicho de desenvolvedores. Nele o usuário faz o cadastro e com esse cadastro o mesmo consegue acessar os serviços do sistema. Como: ver e se candidatar a um evento. Ver livros em pdf. Ver e se candidatar a cursos.
